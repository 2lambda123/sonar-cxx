Legacy code often use primitive types which shall be avoided or limited to platform APIs. C++ types shall be used instead.

  • DEPRECATED: Internal static code analysis checks will no longer be supported in future versions. Plug-in is able to import analysis reports from various external tools. Please consider using external code analyzers instead of deprecated internal ones.

Avoid using the following types:

 WORD  -> use unsigned short instead
 DWORD -> use unsigned long instead
 BYTE  -> use unsigned char instead
 FLOAT -> use float instead
 BOOL  -> use bool instead

Avoid using macros for well known values:

 NULL  -> use nullptr instead
 TRUE  -> use true instead
 FALSE -> use false instead
