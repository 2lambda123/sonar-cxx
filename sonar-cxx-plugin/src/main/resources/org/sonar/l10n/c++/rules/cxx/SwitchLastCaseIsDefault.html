The requirement for a final default clause is defensive programming. This clause should either take appropriate action or contain a suitable comment as to why no action is taken.

  • DEPRECATED: Internal static code analysis checks will no longer be supported in future versions. Plug-in is able to import analysis reports from various external tools. Please consider using external code analyzers instead of deprecated internal ones.

The following code snippet illustrates this rule: 

switch (state) {                       // Non-Compliant - must have a default case
  case 0:
  case 1:
    printf("0 or 1!");
    break;
}

switch (state) {
  default:                             // Non-Compliant - must be last for better readability
    printf("Default");
    break;
  case 0:
  case 1:
    printf("0 or 1!");
    break;
}

switch (state) {
  case 0:
  case 1:
    printf("0 or 1!");
    break;
  default:                             // Compliant
    printf("Default");
    break;
